## 🚀 快速部署步骤

### 必须完成的步骤：

1. **创建 GitHub 仓库**
   - 登录 GitHub
   - 创建新仓库（Public）
   - 上传本项目所有文件

2. **注册 Streamlit Cloud**
   - 访问 https://streamlit.io/cloud
   - 用 GitHub 账号登录

3. **部署应用**
   - 点击 "New app"
   - 选择你的 GitHub 仓库
   - Main file path 填写：`app.py`
   - 点击 "Deploy!"

4. **获取网址**
   - 等待 1-3 分钟部署完成
   - 获得在线网址，格式如：`https://xxx.streamlit.app`
